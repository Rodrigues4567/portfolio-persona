# Portfolio Inspirado em Persona 3

Meu portfolio pessoal inspirado na interface estilizada do jogo Persona 3 Reload, desenvolvido com tecnologias modernas de frontend.

Este projeto foi baseado no repositório original criado pelo usuário(a) Blair Xu, contendo diversas modificações, adaptações responsivas e personalizações visuais e estruturais.

# Funcionalidades

- Interface inspirada em Persona 3 Reload
- Versão responsiva para dispositivos móveis
- Sistema interativo de navegação estilo menu
- Transições e animações
- Seções personalizadas de portfolio
- Design moderno e estilizado
- Conteúdo e informações personalizadas
- Painéis de experiência e resume redesenhados
- Estrutura baseada em componentes

# 📱 Design Responsivo

O projeto foi adaptado para dispositivos móveis, mantendo a identidade visual inspirada em Persona 3 e preservando a experiência original da interface.
